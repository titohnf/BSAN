# BSAN - Sistem Admin POKJA
 
Project Next.js untuk manajemen data POKJA (Kelompok Kerja).

## Cara Menjalankan

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`

## Akun Login

| Role | Username | Password |
|------|----------|----------|
| Admin Pusat | `admin.pusat` | `pusat123` |
| Admin Dinas | `admin.dinas` | `dinas123` |
| Admin Sekolah | `admin.sekolah` | `sekolah123` |

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
- Radix UI
